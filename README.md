## 📌 Summary

**노인들이 쉽게 사용할 수 있는 만보기 앱**

- 노인들이 사용할 수 있도록 만든 만보기앱.

<br />

## 🤔 Background

최근 고령화 현상으로 인해 노인들의 인구가 늘어남에 따라 노인을 위한 앱을 만들어 보겠다 생각했다.

<br />

## 🧐 다른 앱들과의 차이점

1. 기본 18pt ~ 24pt 로 고정하였다.
2. 버튼의 크기를 키워 잘 가독성이 좋게 만들었다.
3. 네비게이션의 깊이를 줄여 앱의 모든 기능을 쓸 수 있게 제작하였다.

<br />
